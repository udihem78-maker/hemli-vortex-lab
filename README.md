## Hemli Vortex Lab

Hemli Vortex Lab is a research-oriented toolkit for analyzing vortex and flow videos using a structured Hemli E/O/S/HP framework.

The project converts video-based flow observations into relative velocity fields, vortex diagnostics, energy spectra, fractal indicators, and Hemli indices:

* **HE — Energy / Result**
* **HO — Order**
* **HS — Stability**
* **HP — Hemli Point / overall balance**

The repository is designed to support a controlled research workflow:

1. Source registration
2. Experiment metadata
3. Calibration and quality scoring
4. Optical-flow-based video analysis
5. Vorticity, Q-criterion, swirl strength, and energy spectrum extraction
6. Hemli index calculation
7. Validation, sensitivity checks, and clean archiving

The project distinguishes between demonstration-level videos, comparative analysis, semi-quantitative results, and research-grade calibrated experiments.

This repository does not store large raw videos or heavy flow-field outputs by default. It stores code, templates, metadata schemas, lightweight result summaries, documentation, and experiment-management tools.
